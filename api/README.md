## Description

Basic API setup
 * /auth <br> authentication / authorization endpoint
 * /assistant <br> endpoint collection for OpenAI assistant features
 * /assistant/shortcut <br> abstraction of OpenAI assistant beta specifically for usage with iOS shortcuts

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
